vector-database-builder — MCP Tool Spec

vector-database-builder

A tool that builds vector databases for Cloudflare.

Overview

This tool creates a vector database for Cloudflare using the Geohash algorithm. It takes in a list of coordinates and returns a vector database with the corresponding Geohash values.

Tools

Geohash

The Geohash algorithm is used to create a compact string representation of a geographic location.

Input Schema


{
  "type": "object",
  "properties": {
    "coordinates": {
      "type": "array",
      "items": {
        "type": "number"
      }
    }
  }
}

Output

A string representing the Geohash value.

Example


{
  "coordinates": [37.7749, -122.4194]
}

Vector Database

A vector database is a data structure that stores a collection of vectors, each representing a geographic location.

Input Schema


{
  "type": "object",
  "properties": {
    "coordinates": {
      "type": "array",
      "items": {
        "type": "object",
        "properties": {
          "latitude": {
            "type": "number"
          },
          "longitude": {
            "type": "number"
          }
        }
      }
    }
  }
}

Output

A vector database with the corresponding Geohash values.

Example


{
  "coordinates": [
    {
      "latitude": 37.7749,
      "longitude": -122.4194
    },
    {
      "latitude": 37.7859,
      "longitude": -122.4364
    }
  ]
}
