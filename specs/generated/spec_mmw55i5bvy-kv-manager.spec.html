kv_manager — MCP Tool Spec

kv_manager

A tool for managing Cloudflare KV keys.

Overview

This tool allows you to list and delete Cloudflare KV keys. It provides a simple interface for managing your KV store.

Tools

list_kv_keys

List all KV keys in your account.

Input Schema


{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "List KV Keys Input",
  "type": "object",
  "properties": {
    "account_id": {"type": "string"}
  },
  "required": ["account_id"]
}

Output

A list of KV keys in your account.

Example


[
  {"key": "key1", "value": "value1"},
  {"key": "key2", "value": "value2"}
]

delete_kv_key

Delete a KV key from your account.

Input Schema


{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "Delete KV Key Input",
  "type": "object",
  "properties": {
    "account_id": {"type": "string"},
    "key": {"type": "string"}
  },
  "required": ["account_id", "key"]
}

Output

A success message if the key was deleted.

Example


{
  "message": "Key deleted successfully"
}

Cloudflare Worker

Runtime: Cloudflare Workers (vanilla JS)

Bindings: KV, Workers KV

Route: /mcp

Deployment

  1. Install wrangler CLI: `npm install -g wrangler`
  2. Log in to Cloudflare: `wrangler login`
  3. Deploy the worker: `wrangler publish`

Connector URL

Format: https://kv-manager.agentfeedoptimization.com/mcp

Suggested name: kv-manager
